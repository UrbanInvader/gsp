# gsp

Google cloud project switcher.

## requirements:

- [gcloud sdk](https://cloud.google.com/sdk/)
- [fzf](https://github.com/junegunn/fzf)

## usage:

`gsp` - list and select from all

`gsp <project id>` - filter based on grep
